## Atividade Angular

Este projeto é uma aplicação feita com **Angular**, com o objetivo de compreender conceitos de componentes, módulos e estrutura básica de uma aplicação Angular. <br/>
Atividade do módulo **Desenvolvimento em Javascript**, do curso **Ciência da Computação**.
